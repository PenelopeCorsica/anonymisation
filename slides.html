Réidentification

  • Sweeney (2001) a réidentifié une base de données médicale pseudonymisée en la croisant avec une liste électorale.
  • Le croisement réalisé sur un triplet de valeurs : code postal, date de naissance et sexe
  • Ce triplet est unique pour environ 80% de la population des Etats-Unis.

Pseudonymisation

Pseudonymisation : traitement de données à caractère personnel de manière à ce qu’on ne puisse pas attribuer les données à une personne physique sans avoir recours à des informations supplémentaires.

Définitions

  • Variables directement identifiantes : nom, prénom, identifiant utilisateur, numéro de Sécurité Sociale, etc.
  • Variables quasi-identifiantes : une variable est quasi-identifiante, si, une fois combinée avec d’autres variables, elle permet l’identification d’un individu. Par exemple, âge, lieu de naissance et métier.
  • Variables sensibles : une variable est dite sensible si elle révèle une information personnelle, privée ou confidentielle à propos d’un individu. Exemple : la personne est malade ou non.

La k-anonymisation

Un jeu de données est k-anonymisé, si et seulement si à chaque combinaison de catégories quasi-identifiantes qui composent la base correspond un minimum de k individus.
